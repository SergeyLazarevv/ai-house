Служебный файл: не попадает в промпт (не совпадает с шаблоном [0-9][0-9]_*.txt).

Сборка system для агента логов:
  1) Файлы prompt/[0-9][0-9]_*.txt — короткая роль и инварианты домена.
  2) Код агента добавляет: краткий формат TOOL_CALL + каталог инструментов, полученный из Graylog MCP server через list_tools.

При изменении аргументов инструмента правьте **app/mcp_servers/graylog.py**. Не дублируйте схему вручную в prompt/.

Код рядом с prompt/: **agent.py** (цикл LLM↔MCP), **system_prompt.py** (сборка system), **parse.py**, **responses.py**, **inputs.py**.
